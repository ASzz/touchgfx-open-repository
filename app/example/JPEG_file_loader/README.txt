1. Introduction
This example shows how to display a JPEG in TouchGFX
The example uses the dynamic bitmap feature which is introduced in TouchGFX release 4.5.0

2. LibJPEG
The example uses LibJPEG which is included in the source.
Use of LibJPEG is subject to conditions.
Read the README file provided with LibJPEG before use.

3. Building and running
A Makefile is provided for the windows simulator.

Build the example in the example directory:
make -f simulator/gcc/Makefile

Run the simulator:
./build/bin/simulator.exe

The file image.jpg must be present in the working directory.

Solution for Visual Studio 2012 is also provided.
